# URL Shortener

Application fullstack de raccourcissement de liens avec tracking de clics et dashboard de statistiques.

## Stack
- **Backend** : Node.js · Express · PostgreSQL · Prisma · JWT
- **Frontend** : Angular

## Fonctionnalités
- Créer des liens courts personnalisés
- Redirection HTTP vers l'URL originale
- Tracking des clics (date, IP, user-agent)
- Dashboard de statistiques par lien
- Authentification JWT

## Lancer le projet

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
ng serve
